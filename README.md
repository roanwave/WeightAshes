# WeightAshes

A personal writing IDE for "The Weight of Ashes," a dark fantasy military epic. This local web application provides NovelCrafter-like functionality for AI-assisted drafting, featuring a Codex knowledge base for worldbuilding, hierarchical manuscript management, intelligent context assembly that automatically injects relevant lore into AI prompts, and multi-provider AI integration.

## Tech Stack

- **Backend:** FastAPI (Python 3.10+)
- **Frontend:** React 18 + Vite + Tailwind CSS
- **Storage:** JSON + Markdown files
- **AI:** Anthropic Claude (primary), OpenRouter/OpenAI (secondary)

## Setup Instructions

*To be filled in as the application is built.*

1. Clone the repository
2. Copy `.env.example` to `.env` and add your API keys
3. Install backend dependencies
4. Install frontend dependencies
5. Run the application

## Note

This is a personal project for a single user's writing workflow. It is not intended for public use or distribution.
